# دليل الأوامر الخاصة بالتيرمينال (Terminal Commands)

هذا الملف يحتوي على جميع الأوامر التي ستحتاجها للتعامل مع المشروع وتحديث الكود. تأكد دائماً أنك تقوم بتشغيل هذه الأوامر وأنت داخل مجلد الكود (`SRC`).

## 1. أوامر التحديث والتشغيل (Compilation & Execution)

* **لعمل تحديث (Update) للكود:**
  هذا الأمر تقوم بكتابته بعد أي تعديل تقوم به على ملفات الـ `cpp`، لكي يتم ترجمة وتحديث ملف الـ `exe` ليحتوي على أحدث كود.
  ```powershell
  cmd /c "g++ *.cpp -o HospitalSystem.exe"
  ```

* **لتشغيل البرنامج (بعد عمل التحديث):**
  ```powershell
  .\HospitalSystem.exe
  ```

* **لعمل تحديث (Update) وتشغيل في نفس الوقت (بأمر واحد):**
  هذا هو أفضل وأسهل أمر تستخدمه دائماً لتوفير الوقت.
  ```powershell
  cmd /c "g++ *.cpp -o HospitalSystem.exe" ; .\HospitalSystem.exe
  ```

## 2. أوامر التنقل (Navigation)

* **للدخول إلى المجلد الذي يحتوي على ملفات الكود (`SRC`):**
  (وهذه الخطوة مهمة جداً قبل تشغيل أي أمر من أوامر التحديث)
  ```powershell
  cd SRC
  ```

* **للرجوع خطوة للخلف (الخروج من مجلد الـ SRC):**
  ```powershell
  cd ..
  ```

* **لعرض كل الملفات الموجودة في المجلد الذي تقف فيه الآن:**
  ```powershell
  ls
  ```

## 3. أوامر إدارة الملفات وتنظيف الشاشة (File Management)

* **لتنظيف الشاشة من الكلام الكثير أو الأخطاء:**
  ```powershell
  clear
  ```

* **لحذف البرنامج القديم (ملف الـ exe) يدوياً:**
  ```powershell
  del HospitalSystem.exe
  ```

* **لحذف جميع بيانات الأطباء والمرضى (إذا أردت بدء البرنامج من الصفر):**
  هذا الأمر يقوم بمسح ملفات الـ Text لتبدأ النظام كأنه جديد.
  ```powershell
  del doctors.txt
  del patients.txt
  ```
